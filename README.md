# interest-calculator
Sample code for AEP 2017 - CI 101

This project supports both Maven & Ant as build tools.

## Ant instructions

To run tests

```
ant
```

To run the build (clean and test)

```
ant clean test
```

## Maven

To run tests:

```
mvn test
```

To run the build (clean and build):

```
mvn clean install
```
